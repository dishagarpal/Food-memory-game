# Food Memory Game

A fun browser-based memory matching game where players flip cards to match food pairs.

##  Features

* Interactive card flipping
* Match food pairs
* Simple and responsive UI

## Tech Stack

* HTML
* CSS
* JavaScript

## How to Run

Open `index.html` in your browser
